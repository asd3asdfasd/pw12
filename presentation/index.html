01 Введение

Актуальность, цель и задачи

Актуальность. В 2024–2026 гг. рынок пожарной автоматики РФ трансформировался: запрет параллельного импорта (приказ Минпромторга №4611), технологический сбор с 01.09.2026 и ужесточение ГОСТ Р 53325-2024 / СП 484 сформировали защищённую нишу для отечественного производителя.
Цель

Разработать инвестиционный проект малого предприятия по выпуску комбинированных пожарных извещателей и оценить его экономическую эффективность.

Задачи
  • Обосновать технико-экономическую целесообразность на основе анализа рынка
  • Определить показатели эффективности и выбрать инструмент моделирования
  • Разработать производственную и организационную модель с автоматизацией контроля качества
  • Построить финансовую модель в Project Expert и провести анализ чувствительности

06 Проектная часть · Автоматизация

Автоматизированный контроль качества: стенд + MES

  • Стенд «СИ-ТДК» — аэродинамический дымовой канал + климатическая камера теплового канала
  • 100% выходной контроль каждого изделия вместо выборочного 1–5%
  • MES-система обеспечивает сквозную прослеживаемость по серийному номеру
  • Аппаратная блокировка печати этикетки для бракованного изделия — брак физически не выйдет за цех
Ключевое преимущество для защиты: автоматизация контроля исключает человеческий фактор и репутационные риски (штрафы МЧС за ложные срабатывания на объектах Ф1.1, Ф4.1).
